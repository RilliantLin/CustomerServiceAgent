本手册介绍智能音箱 X1 的安装、使用与常见故障排查，适用于固件 v2.0 及以上版本。

# 快速开始

收到设备后，请先检查包装内是否包含：主机、电源适配器、快速指南各一份。

## 通电与配网

1. 将电源适配器接入主机底部接口并插电。
2. 听到「叮咚」提示音后，打开手机 App。
3. 进入「添加设备」，按提示完成 Wi-Fi 配网。

配网时请确保手机与音箱处于同一 2.4GHz 网络，暂不支持 5GHz 频段。

## 语音唤醒

默认唤醒词为「你好小X」。可在 App 的设置中自定义唤醒词，唤醒词需为 3-5 个汉字。

# 常见问题

## 无法配网

请按以下顺序排查：

- 确认输入的 Wi-Fi 密码正确。
- 确认路由器开启了 2.4GHz 频段。
- 长按主机顶部按键 5 秒恢复出厂设置后重试。

如果仍然失败，可通过命令行查看设备日志：

```sh
# 下面这一行以 # 开头，但它在代码块内，不应被当作标题
adb logcat | grep wifi
```

## 没有声音

检查是否处于静音模式，音量是否调至最低。若指示灯正常但无声音，可能是扬声器故障，请联系售后。

# 保修与售后

整机保修一年，主板核心部件保修两年。人为损坏、进水、私自拆机不在保修范围内。保修期内的非人为故障可享受免费维修或换新。
